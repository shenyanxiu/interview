# 什么是 Agent

## 问题

什么是 Agent？

## 主题

AI 基础概念

## 难度

初级

## 参考答案

Agent 是一种基于大语言模型（LLM）的智能体架构，核心特征是：**模型不仅生成文本，还能自主决策、调用工具、执行多步操作来完成目标。**

关键组成：

- **LLM 作为推理核心** — 理解意图、规划步骤、判断下一步行动
- **工具调用能力** — 读写文件、执行命令、搜索信息、调用 API 等
- **循环执行** — 观察结果 → 思考 → 行动 → 再观察，直到任务完成
- **自主性** — 不需要每一步都等人类指令，能根据中间结果自行调整策略

和普通聊天机器人的区别：聊天机器人只做一轮问答，Agent 能持续行动直到把事情做完。比如你说"修复这个 bug"，Agent 会自己读代码、定位问题、改代码、跑测试、验证结果，而不是只告诉你"你可以试试改第 42 行"。

## 易错点

- 把 Agent 等同于聊天机器人，忽略了"自主行动"这个核心区别
- 认为 Agent 不需要人类监督，实际上大多数 Agent 仍需要人类确认关键决策

## 知识延伸

- ReAct 模式（Reasoning + Acting）
- 多 Agent 协作架构
- Agent 的记忆机制（短期/长期记忆）
- Tool Use / Function Calling
